# WeSki projects

## Development

```
cd weski/
npm i && npm run serve:all
```

If browser doesn't open - visit http://localhost:4200/

## Storybook

Browse available components (incl. Chakra-UI) by running

```
npm run storybook
```

If browser doesn't open - visit http://localhost:4400/
